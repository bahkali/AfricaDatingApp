# PROFILE SERVICE

This is the profile service provide the basic user information like his bio..
user can update their information.

## Feature

this project include

- REST API CRUD
- Repository Pattern
- Cloudinary to Store our photo
- Entity Framework
- Sqlite Db
- Docker

## Main User Case

- Get Profile with user id and Username
- Create Profile
- Update Profile
- Add Photo
- set Main Photo
- Delete Photo
